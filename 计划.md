# Recaping - 开发计划

## 项目概述

Recaping 是一款多媒体录音记录应用，允许用户在录音过程中同时记录照片、视频片段和文字笔记，形成完整的多媒体时间轴记录。

### 核心功能

1. **录音功能**：持续录音，自动分段存储
2. **多媒体记录**：照片、短视频片段、文字笔记
3. **时间轴展示**：双时间轴模型
4. **回放功能**：同步回放录音和多媒体内容
5. **AI 功能**：转文字、智能摘要、关键词提取

### 技术栈

- **框架**：Flutter 3.x + Dart
- **数据库**：sqflite（SQLite）
- **音频录制**：record
- **音频播放**：just_audio
- **相机**：image_picker
- **状态管理**：flutter_riverpod
- **路由**：go_router
